Project Overview

Designed and implemented a 12-bit 100MS/s pipeline Analog-to-Digital Converter (ADC) in 65nm CMOS technology. The ADC achieves 68dB SNDR while consuming 24mW from a 1.2V supply.

Key Specifications

  • Resolution: 12-bit
  • Sampling Rate: 100MS/s
  • Supply Voltage: 1.2V ±10%
  • Power Consumption: 24mW
  • ENOB: 10.9 bits @ Nyquist

Key Features

  • 1.5-bit/stage pipeline architecture
  • Bootstrapped sampling switch
  • Current-mode MDAC with gain-boosted opamps
  • On-chip bandgap reference

Architecture

The pipeline architecture was selected for optimal speed/power tradeoff. The 1.5-bit/stage approach with digital error correction relaxes comparator precision requirements while maintaining good linearity.

Pipeline ADC block diagram showing 12-stage implementation

Stage Breakdown

  • Front-end S/H with bootstrapping
  • 10x 1.5-bit pipeline stages
  • Final 2-bit flash stage
  • Digital error correction

Advantages

  • Reduced comparator count vs flash
  • Relaxed amplifier settling requirements
  • Digital correction of analog errors
  • Scalable for higher resolutions

Analog Circuit Design

The analog front-end includes a bootstrapped sampling switch followed by a residue amplifier with gain-boosted cascode topology, achieving >90dB DC gain.

Gain-boosted residue amplifier schematic

Simulated AC response (89dB gain, 800MHz UGBW)

Key Circuits

  • Bootstrapped switch: Charge pump maintains constant Vgs for linear sampling
  • Residue amplifier: Two-stage with gain boosting for high DC accuracy
  • Dynamic comparator: Auto-zeroing to reduce offset
  • Reference buffer: Current-mode for fast settling

Layout Implementation

Careful attention to matching, parasitics, and signal integrity resulted in a compact 2.1mm × 1.8mm implementation.

Full chip layout showing analog and digital partitions

Analog Techniques

  • Common-centroid capacitor arrays
  • Guard rings for sensitive circuits
  • Separate power domains
  • Shielded clock distribution

Digital Integration

  • Standard cell implementation
  • LVDS output drivers
  • SPI configuration interface
  • ESD protection on all I/Os

Measurement Results

Post-silicon verification showed the ADC met all specifications with margin, achieving state-of-the-art performance for its power class.

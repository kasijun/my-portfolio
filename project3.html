VHDL-Design of a RISC Processor Core

Design, implement and verify a custom Reduced Instruction Set Computer (RISC) to execute a given set of software instructions. The processor core uses a ROM (Read Only Memory) to fetch instructions and a RAM (Random Access Memory) to store results and operands.

Project Overview

Estimate a circuit’s power consumption and to evaluate power saving measures, the actual power consumption has to be analyzed at all different stages in the design flow. The methods used have to take into account the available relevant parameters for power consumption at each stage of the design flow. For a high level digital design flow, there are three dominant parameters:  Switching activity of a net  intrinsic power consumption of a gate  parasitic capacitance of a net These parameters have to be refined to better accuracy during the flow from RTL VHDL code to a chip layout (GDS). To do so and to actually calculate and analyze the power consumption, different data formats and different tools are used at each level
